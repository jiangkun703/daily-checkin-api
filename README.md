# 每日签到API

基于星座运势API架构开发的每日签到系统，专为微信插件设计。

## 功能特点

- ✅ **每日签到**：每个用户每天只能签到一次（基于伪随机算法）
- ✅ **连续签到统计**：记录连续签到天数
- ✅ **积分奖励系统**：基础积分 + 连续签到额外奖励
- ✅ **幸运元素**：每日随机幸运物品、颜色、数字
- ✅ **微信兼容**：直接返回格式化文本，微信可直接显示
- ✅ **无需数据库**：基于日期的伪随机算法，同一天同一用户结果一致

## API使用

### 基本签到
```
GET /api?msg=签到&userId=user001&userName=爸爸
```

### 参数说明
| 参数 | 必填 | 说明 |
|------|------|------|
| msg | 是 | 固定为"签到"、"打卡"或包含"签"字 |
| userId | 否 | 用户ID，默认"user001" |
| userName | 否 | 用户名，默认"用户" |

## 技术原理

### 伪随机算法
基于用户ID和日期的伪随机数生成，确保：
- 同一天同一用户签到结果一致
- 不同用户或不同日期结果不同
- 无需数据库存储，轻量高效

### 连续天数计算
简化实现：基于用户ID和日期的伪随机算法生成1-30天的"连续签到"。
实际应用中可替换为数据库查询。
